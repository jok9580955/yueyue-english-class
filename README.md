# 钥钥的英语课

一个面向 2-6 岁宝宝的 iPhone / iPad 英语启蒙 SwiftUI MVP。

## 当前功能

- 分龄课程：2-3 岁、4-5 岁、6 岁
- 主题内容：动物、食物、家庭、颜色、玩具、自然、身体、动作
- 真实插画：24 张统一风格儿童词卡图片
- 离线点读：使用 iOS 系统英文语音
- 跟读鼓励：完成练习后获得星星
- 课堂小游戏：气泡点点、影子配对、节奏拍词、字母小火车
- 故事时间：每个主题配有短句故事，可逐句播放
- 学习记录：今日路径、成就贴纸、一周启蒙计划
- 小测互动：听音后选择对应中文含义
- 家长中心：每日目标、温和模式、语速调节
- 名字设置：默认“钥钥的英语课”，修改宝宝名字后显示“某某的英语课”
- iPad 兼容：大屏自动使用分栏布局

## 运行

在 Xcode 中打开 `EnglishSprout.xcodeproj`，选择 iPhone 或 iPad 模拟器运行。
